# AI-Powered E-Commerce Analytics Platform

A full-stack analytics platform with ML predictions and natural language querying built on real e-commerce data.

## Tech Stack
- **Backend:** Python, FastAPI, Pandas, NumPy, Scikit-learn
- **Databases:** MongoDB Atlas, PostgreSQL (Supabase)
- **AI/RAG:** LangChain, Groq LLM
- **Frontend:** React, TailwindCSS, Recharts
- **DevOps:** Docker, GitHub Actions CI/CD

## Features
- Real-time analytics dashboard with 100K+ orders
- ML models for customer churn and revenue prediction
- Natural language querying via RAG
- Paginated orders table
- CI/CD pipeline with Docker
